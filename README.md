# Basic MLops things
A training Python project demonstrating the first steps toward MLOps: neural network training, saving model weights to `model.pth`, loading the trained model for inference without retraining, serving predictions through a Flask API, and performing automated architecture and hyperparameter search.

The project also demonstrates experiment reproducibility, metric comparison, early stopping and result caching. These concepts can later be extended with MLflow for experiment tracking, model versioning and model registry management.
